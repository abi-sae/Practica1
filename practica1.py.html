""" 
Calcular el impuesto sobre la renta (ISR) anual de una persona. 
 
Los datos de entrada son: 
Sueldo bruto anual (s) y número de dependientes (d). 
 
Requerimientos: 
1. A todos los contribuyentes se les permite una deducción estándar de $10,000. 
2. A todos los contribuyentes se les cobra una tasa de impuestos fija del 20%. 
3. Por cada dependiente, al contribuyente se le permite una deducción adicional de $2,000. 
4. El sueldo bruto debe ingresarse al centavo más cercano. 
5. El ISR se expresa como un número decimal. 
 
 
Calculos 
ISR=(Sueldo bruto -(Número de dependientes*2000))*0.20 
 
Datos de salida 
Total a pagar de ISR 
 
Nota: Si se introduce algún número de acuerdo al salario bruto menor a 10000, el programa aún así realizara las funciones que tiene, 
sin embargo el resultado no es verdadero, pues se necesitan minimo 10000 para que de datos correctos. 
 
"""
#Se piden los datos de entrada.
Sueldo=float(input("Introduce el sueldo burto anual:"))
Dependientes=float(input("Introduce el número de dependientes:"))

#Se realizan los calculos correspondientes de acuerdo a los datos recolectados.
ISR=(Sueldo-(Dependientes*2000))*0.20

#Aquí se utiliza el comando para redondear los decimales, que se tenga los 2 mas cercanos.
isr= round(ISR,2)

#Se despliegan los resultados.
print("El total a pagar de el impuesto sobre la renta: es",isr, )
